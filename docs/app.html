Model at a Glance

ImportantHonest limitations

pct_games_missed captures all absence reasons — Achilles injuries, other injuries, rest decisions, and suspensions. The model predicts load-related game unavailability as a proxy for injury burden. It does not diagnose specific injury types.

back_to_back_games and total_minutes were removed from the model after PDP analysis revealed survivorship bias — healthy players accumulate more of both, so the features had inverted relationships with the outcome. The primary load predictor is avg_rolling_minutes_3 (lagged 3-game rolling average minutes), which correctly shows a positive monotonic relationship with games missed.

Full methodology in the academic paper.

## Honest limitations
`pct_games_missed` captures **all** absence reasons — Achilles injuries, other injuries,
rest decisions, and suspensions. The model predicts load-related game unavailability
as a proxy for injury burden. It does not diagnose specific injury types.

`back_to_back_games` and `total_minutes` were removed from the model after PDP
analysis revealed survivorship bias — healthy players accumulate more of both,
so the features had inverted relationships with the outcome. The primary load
predictor is `avg_rolling_minutes_3` (lagged 3-game rolling average minutes),
which correctly shows a positive monotonic relationship with games missed.

Full methodology in the [academic paper](submission.qmd).
:::
